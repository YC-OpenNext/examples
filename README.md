# YC-OpenNext Examples

Example Next.js projects demonstrating deployment to Yandex Cloud with [YC-OpenNext](https://github.com/YC-OpenNext/cli).

## Examples

| Directory | Next.js | Router | Features |
|-----------|---------|--------|----------|
| `next12-pages-isr/` | 12.3.7 | Pages | ISR, catch-all routes, API routes, i18n |
| `next13-app-router/` | 13.5.11 | App | Server Components, parallel routes, streaming |
| `next14-mixed/` | 14.2.35 | App + Pages | Middleware, Server Actions, mixed routing |
| `next15-modern/` | 15.5.12 | App | React 19, Server Actions, partial prerendering |

## Getting Started

```bash
cd next14-mixed
npm install
npm run build
npm run dev
```
